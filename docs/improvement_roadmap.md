# Paths to a higher honest ceiling — researched improvement roadmap

*Evidence-grounded analysis (multi-agent literature sweep + adversarial reality check) of which levers would raise the HONEST (family-holdout) ROC/PR for genome-based zoonotic prediction, and by how much. Baseline = frozen ESM-2 15B + probe: family-holdout ROC 0.757, PR-AUC 0.293. Companion to PAPER.md §11.1.*

All key numbers verified against the repo. The numbers match the reality check (ESM-15B vs effort_only at family holdout: ROC +0.052, PR −0.124, p=0.99; leakage curve 0.893→0.669 with phylogenetic distance for ESM, 0.916→0.635 for composition). Note one refinement: the canonical contrast uses `composition_xgb` vs `family_prior` for the 0.234 ROC gap, and the `esm_xgb_vs_composition_xgb__family` contrast is +0.024 ROC / +0.002 PR (both CI-crossing-zero), which is itself decisive evidence for the "near-exhausted" claim.

## Paths to a higher honest ceiling

Every result in this paper is reported under family-level holdout, where whole viral families are removed from training. This is the only split that estimates the quantity we care about — performance on a virus unlike anything seen before — and it is brutal: our best model (frozen ESM-2 15B + XGBoost) reaches family-holdout ROC **0.757** and PR-AUC **0.293**, against a base rate of 0.117. The leakage curve makes the reason explicit: ESM model skill decays monotonically with phylogenetic distance to the training set, from AUC **0.893** in the nearest-neighbor bin to **0.669** at mid-distance (composition: 0.916 → 0.635). Almost all apparent skill is distance-to-training, i.e. phylogeny. Two facts must frame any roadmap. First, **the frozen 15B does not significantly beat a one-line host-generalism/effort baseline on the honest split**: ESM-vs-`effort_only` at family holdout is ROC +0.052 (95% CI −0.137 to +0.216, p=0.37) and PR **−0.124** (CI −0.258 to −0.019, p=0.99) — the genome *loses* on PR. Second, the 15B's edge over a simple genome-composition model is also gone (ROC +0.024, CI −0.060 to +0.103, p=0.36; PR +0.002, p=0.49). The contrast CIs are **±0.05–0.10 wide**, so any point estimate below ~+0.05 ROC / ~+0.05 PR is statistically invisible in this harness. We assess six levers against this bar. Throughout, we never read random-split or in-distribution gains as family-holdout gains — that conflation is the single most common way these numbers get inflated.

### 2. The top three levers, argued

**Lever 6 (fusion of genome + generalism + ecology) is the only lever with a mechanistically defensible path to a real honest gain — but only half of it should be built.** The argument is simple and grounded in our own numbers: host-generalism, computed from the *host-association table* and not the genome, already *ties or beats* the 15B genome on the honest split (family-holdout PR 0.303 vs 0.293; ROC 0.631). It is a genuinely orthogonal feature axis that is *available for held-out families*, which is exactly what survives the split. The analog is direct: the Poisot-lab *Patterns* (2023) result that "graph embedding of the imputed host–virus network improves predictions of human infection from viral genome features" — fusing network/host-structure pseudo-traits with a sequence model beats the sequence model alone. The honest catch, which we will not hide, is that generalism and the genome are *near-redundant in aggregate* (the generalism-vs-15B contrast had p≈0.57, CI crossing zero), so they make different errors but carry overlapping information; late fusion of partially-redundant experts typically yields **+0.003 to +0.03 ROC** (Mollentze 2021 saw ~+0.02–0.03 from adding an orthogonal genomic feature set). That is why our point estimate is +0.02 ROC / +0.03 PR, not the +0.04 the optimistic case suggests. Crucially, **two components must be cut**: phylogenetic eigenvectors (our `tax_family` prior is ROC **0.42** — *anti-predictive* — so eigenvectors extrapolate garbage for held-out families) and effort/citation-style traits (sampling-bias confounds that inflate the random split and evaporate or hurt under holdout, exactly the leakage the split is designed to strip).

**Lever 5 (PU learning + genome-external propensity correction) is the right companion to fusion — a label-hygiene lever that adds a sliver of honest PR and keeps the evaluation defensible.** Our labels are positive-unlabeled: a virus is "negative" partly because nobody looked. The legitimate, family-transcending version reweights the empirical risk using a propensity model built *only* from genome-external covariates that exist for held-out families (discovery year, GenBank record count, host-order surveillance intensity). The closest methodological analog — PU-GO, which applies PU learning on top of ESM-2 embeddings — added **+0.018 AUPR on average** on the harder similarity-based split, but went *negative* (−0.0045) on one of three ontologies, which bounds the realistic upside and warns of instability. The decisive caveat from our own data: the effort/observation signal is **already near-saturated by the one-line generalism baseline** (effort_only family PR 0.3035), so a propensity correction is mostly re-deriving signal we already have rather than adding new family-transcending signal. Honest expectation: ROC ~0, PR +0.01–0.02 *if and only if* the propensity is strictly genome-external and held out family-wise. The +0.05 PR tail is wishful, and the naive "add effort as a feature" variant is re-leakage, not a lever.

**Lever 3a (ESM-C / ESM-3 swap) is worth doing for cost, not for ceiling.** Our ladder is still rising (3B→15B ROC 0.708→0.757 on the `xgboost` probe), and ESM-C 600M matches ESM-2 3B while ESM-C 6B matches/beats ESM-2 15B on contact prediction — so adopting ESM-C is a cheaper way to occupy the same rung, with a small chance of +0.01 ROC from marginally better convergent-feature representation. But it will not clear the ±0.05 CI: "PLMs are Accidental Taxonomists" shows ESM-C encodes taxonomy at ~0.87 F1, *no less* than ESM-2, so a newer PLM does not reduce the phylogenetic leakage that bounds the honest number — it relabels it. We explicitly **do not** recommend spending 1–2 GPU-weeks on structure tokens (3b): FLIP shows SaProt regresses under distribution shift (GB1 low→high Spearman 0.362 vs ESM-2 0.535), and predicted structures are least reliable for precisely the proteins that decide human tropism.

### 3. The realistic reachable honest ceiling

The levers **do not stack additively**. Levers 1–4 all re-weight the *same* near-exhausted family-transcending signal in the frozen genome; lever 5 cleans labels; only lever 6 adds new (off-genome) information. Their honest deltas should be treated as strongly sub-additive. Anchoring on the evidence that the 15B already sits at the genome-composition ceiling (Mollentze 2021 composition AUC ~0.773; our 15B-vs-composition contrast is null), our best-guess for a determined, fully honest effort is:

- **Family-holdout ROC ≈ 0.78 (range 0.77–0.79)** — roughly +0.02–0.03 over 0.757. Almost all of it comes from fusion importing generalism/ecology; PLM and input levers contribute single-digit-thousandths each and are individually CI-invisible.
- **Family-holdout PR-AUC ≈ 0.33–0.35 (range 0.31–0.37)** — roughly +0.02–0.06 over 0.293. Fusion does the heavy lifting (the goal is to robustly clear the generalism PR of 0.303, which 15B only ties); PU correction adds a sliver if strictly external.

Both numbers carry CI half-widths of **±0.05**. The brutally honest statement is therefore: the ceiling we can *demonstrate to be significantly above the one-line host-range baseline* is **at best ROC ≈ 0.79, PR ≈ 0.35, and even that is borderline-significant**. A determined effort whose 95% CI cleanly excludes the generalism baseline is not guaranteed to exist.

**Where the ecological ceiling caps this.** None of these levers can pass the hard information bound: spillover risk is *partly ecological* — exposure, contact rate, geographic sympatry, reservoir overlap — and that information is simply **not in the genome**. Levers 1–4 can only re-weight what the sequence already encodes (a cleaner deterministic function of the genome is still a phylogenetic coordinate); lever 5 only de-biases labels; lever 6 imports a *sliver* of off-genome ecology but is itself capped because VIRION's ecological features are coarse, sampling-biased, and partly correlated with taxonomy. The leakage curve is the empirical signature of this wall: skill is distance-to-training, and once a held-out family is far from anything seen, the genome retains little. The remaining gap to perfect prediction is genuinely off-genome exposure/contact information that no model in *this* data pipeline can recover. This is a ceiling result, not a leaderboard result.

### 4. Best bang-for-buck given we already have frozen-15B embeddings

Given the embeddings are already cached, **the single best investment is Lever 6 fusion, restricted to host-generalism plus a small set of clean ecological/trait features (transmission mode, envelope status, reservoir order, host-range breadth), trained as an out-of-fold calibrated stacker inside the existing family-clustered CV harness.** It is the cheapest lever in the set (~1–2 days, no GPU, all ingredients already on the cohort), and it is the *only* one that adds information the frozen genome physically cannot carry — making it the only lever with a defensible route past the PR 0.303 generalism baseline that the 15B merely ties. The frozen embeddings turn what would be an expensive program into a one-script experiment: the 15B out-of-fold probe scores already exist, so fusion is a logistic/shallow-XGB meta-learner over 3–4 base scores, not a retraining job. Pair it with Lever 5's genome-external propensity correction (also free, also no GPU) as a label-hygiene companion. Everything that *consumes* GPU — fine-tuning (Lever 1), structure tokens (Lever 3b), full-proteome re-embedding (Lever 4) — buys CI-invisible deltas on the same exhausted genome signal and should be deprioritized; if GPU is spent at all, spend it once swapping to ESM-C (Lever 3a) to *cut* the cost of the existing pipeline, not to chase a ceiling that the frozen 15B has already reached.
